## Getting Started

To get started with this project, follow these steps:

1. Clone the repository
2. Install dependencies with `npm install`
3. Run the development server with `npm run dev`
4. Visit `http://localhost:3000` in your web browser


## Usage

[View Live App](https://ai-image-generator-ncgd5ws5d-motypes.vercel.app/)

To use the image generator, simply enter a text prompt and click "Generate Image". The generator will use DALL路E and GPT-3.5 to create an image based on your input.
